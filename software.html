    Software Tools

    • Aircrack-ng (monitoring, AP/client recon, WEP & WPA2 hacking). Install from repositories (sudo apt-get install aircrack-ng).
    • Reaver (WPS/PixieDust) compile from source https://github.com/t6x/reaver-wps-fork-t6x
    • Mdk3 (brute force hidden networks, stress test Wifi network) repository "sudo apt-get install mdk3"
    • Hcxdumptool/hcxpcaptool (capturing PMKID info) compile:
      https://github.com/ZerBea/hcxdumptool
      https://github.com/ZerBea/hcxtools
    • Hashcat (cracking variety of password hashes, specifically PKMID) Download newest binary here: https://hashcat.net/files/hashcat-5.1.0.7z

    Alternative Tools

    • Cowpatty (WPA2 password cracking, rainbow table generation)
    • Pyrit (GPU, multi-core password cracker & rainbow table generator)
    • Bully (WPS, PixieDust)
    • Airgeddon (Finding insecure networks)
    • Kismet (IDS, AP/Client recon, hidden network detection)
